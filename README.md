A meta-template for bootstrapping new [Copier][1] project scaffolds.

## Usage

1. Install the latest Copier via command:

    ```text
    pip install --upgrade copier copier-template-extensions
    ```

2. Use command `copier` to generate a new template:

    ```text
    copier copy gh:garrett-he/copier-meta /path/to/destination
    ```
